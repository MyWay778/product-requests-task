# Product Requests Task

Web application to view and edit product requests.

### Technology Stack

- Nuxt 4 (Vue3, Composition API)
- TypeScript
- Sass Modules

## How to Run

### Setup

Make sure to install dependencies:

```bash
npm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```
